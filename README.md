
# Student Result Management System

## Project Description
The Student Result Management System is a simple Python program that collects a student's name, matric number, and score, then calculates and displays the student's grade automatically.

This project was developed as part of the **SEN 201 assignment** to demonstrate understanding of the **Software Development Life Cycle (SDLC)** and basic Python programming.

---

## Features
- Stores student name and matric number
- Accepts a student score
- Automatically calculates grade
- Displays student result clearly

---

## Grade System
- 70 – 100 : A  
- 60 – 69  : B  
- 50 – 59  : C  
- 45 – 49  : D  
- Below 45 : F

- Python
